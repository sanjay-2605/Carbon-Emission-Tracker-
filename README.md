# Carbon-Emission-Tracker-
A full-stack climate-tech application for tracking, analyzing, and visualizing carbon emissions across operational activities.

## Features

- Scope 1, 2, and 3 Emission Tracking
- Sustainability Analytics Dashboard
- Data Visualization using Chart.js
- Responsive Frontend
- REST API Integration
- Firebase Cloud Storage

## Tech Stack

Frontend:
- React.js
- Chart.js
- Axios

Backend:
- Node.js
- Express.js

Database:
- Firebase Firestore
